# Aurora Sign Up ✨

Interfaz de registro moderna a dos columnas — hero con video de fondo + formulario.
Construida con **React + Tailwind CSS v4 + motion/react + lucide-react** (Vite + TypeScript).

Todo el diseño vive en `src/App.tsx` y `src/index.css`.

## Correr

```bash
npm install
npm run dev      # http://localhost:5173
```

## Build de producción

```bash
npm run build    # genera dist/
npm run preview
```

## Estructura

```
src/
  App.tsx     # toda la UI: layout 2 columnas + componentes (StepItem, SocialButton, InputGroup)
  index.css   # Inter + Tailwind v4 (@theme: --font-sans, --color-brand-gray)
  main.tsx    # bootstrap de React
```
